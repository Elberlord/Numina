NÚMINA v3.2 — ACTUALIZACIÓN CON SERIE

1. Extrae el ZIP.
2. Copia todos los archivos internos a la raíz del repositorio Numina.
3. No borres la carpeta .git.
4. Acepta reemplazar los archivos existentes.
5. Ejecuta git add ., commit y push.
6. Espera la publicación de GitHub Pages.
7. Cierra completamente la PWA y vuelve a abrirla. Si conserva la versión anterior, desinstálala y vuelve a instalarla.

FIREBASE:
- No cambies las reglas de Firestore. Son las mismas de la versión 3.1.

NOVEDAD:
- Al crear una campaña se elige Sin serie o Con serie.
- Con serie, cada venta y consulta exige número + serie.
- Las campañas antiguas quedan como Sin serie.
