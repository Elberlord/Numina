NÚMINA — SUBIR A GITHUB PAGES

1. Descomprime numina-github-pages-v1.zip.
2. Crea un repositorio nuevo en GitHub.
3. En el repositorio, pulsa Add file > Upload files.
4. Arrastra TODO el contenido descomprimido, incluyendo las carpetas icons y el archivo .nojekyll.
5. Confirma la carga con Commit changes.
6. Abre Settings > Pages.
7. En Source/Build and deployment selecciona Deploy from a branch.
8. Selecciona main y / (root), luego Save.
9. Abre la dirección que GitHub muestre, normalmente:
   https://TU-USUARIO.github.io/NOMBRE-DEL-REPOSITORIO/
10. En Android abre esa dirección con Chrome y elige Instalar aplicación o Agregar a pantalla principal.

IMPORTANTE:
Esta primera versión guarda la información en cada teléfono. No sincroniza ventas entre teléfonos. Sirve para probar la instalación, el modo sin internet y todas las funciones locales. Para compartir una sola base entre vendedores habrá que conectar una base de datos en línea.
